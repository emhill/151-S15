PWOD4: Rectangle Class

In class we saw how to use inheritance to reuse code between our Ellipse and Circle classes. Now it’s your turn.

Instructions

  1. Start your timer
  2. If you haven’t been following along in class, download the Shapes Project and import it into your workspace.
  3. Use inheritance to add a Rectangle class to your Shapes project. First, think about how Square and Rectangle are related. Should Rectangle extend Square, or vice versa? We went over both ways in class.
  4. Determine what fields are needed in the Square & Rectangle classes. Overlapping fields should only be in one class.
  5. Your Rectangle & Square classes should have at least 3 constructors each:
    • Default constructor (no parameters)
    • Constructor with parameters for each field except outlineColor
    • Constructor with with parameters for each field
  6. Add get & set methods for all the fields. There should not be any duplicated get/set methods.
  7. Make sure the getArea() and getPerimeter() methods work for both classes, and test in main by creating at least one Rectangle and one Square object.
  8. Make sure the draw() method works for both classes and test it by adding a Rectangle and Square to your Picture class.
  9. Run the program to verify your changes.
  10. Stop your timer
